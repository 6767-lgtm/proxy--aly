## 📝 Step-by-Step Usage

### **STEP 1: ENTER A URL**

**WHERE TO TYPE:**
- Look at the **TOP LEFT** of the screen
- You'll see a text box that says: "Enter URL (e.g., google.com)"
- **Click on this box** and type your URL

**WHAT TO TYPE (Examples):**
```
google.com
wikipedia.org
reddit.com
youtube.com
example.com
```

**You CAN type:**
- ✅ `google.com` (without https://)
- ✅ `https://google.com` (with https://)
- ✅ `www.google.com`

**ProxyHub automatically adds https:// if you forget it!**

---

### **STEP 2: SELECT A PROXY**

**WHERE TO SELECT:**
- Look at the **dropdown menu** next to the URL box (shows "CORSProxy")
- Click it to see options

**CHOOSE ONE:**
```
☐ CORSProxy (Recommended - Fastest)
☐ AllOrigins (Alternative)
☐ API (AllOrigins) (API Version)
```

**Start with CORSProxy - it works best!**

---

### **STEP 3: CLICK LAUNCH**

**WHERE TO CLICK:**
- Find the **blue "🚀 Launch" button**
- Click it

**WHAT HAPPENS:**
1. You'll see: "⏳ Loading..." at the bottom
2. The website starts loading
3. After a few seconds: "✓ Loaded successfully"
4. The website appears in the main area!

**If it fails:**
- Try a different proxy from the dropdown
- Try a different website

## 📋 COPY URL FEATURE (New!)

### **How to Copy a URL:**

**Method 1: Click the Copy Button**
1. Enter a URL in the input box
2. Click the **📋 Copy** button
3. You see: "✓ URL copied to clipboard!"
4. Paste anywhere with `Ctrl+V`

**Method 2: Keyboard Shortcut**
1. Press **`Ctrl+C`** (Windows/Linux) or **`Cmd+C`** (Mac)
2. Message: "✓ URL copied to clipboard!"
3. Paste with `Ctrl+V` or `Cmd+V`

## ⭐ BOOKMARKS (LEFT SIDEBAR)

**WHAT IS IT?**
Save your favorite websites for quick access later

**HOW TO USE:**
1. Load a website (click Launch)
2. Click **"+ Add Bookmark"** button on the left
3. Type a name for the bookmark (e.g., "My Google")
4. Click the bookmark to load it again
5. Click **✕** to remove a bookmark

**KEYBOARD SHORTCUT:**
- Press **`Ctrl+B`** to add bookmark quickly

---

## 📜 HISTORY (LEFT SIDEBAR)

**WHAT IS IT?**
Automatically tracks websites you visit

**HOW TO USE:**
1. Visit websites (click Launch)
2. See them appear in the History section
3. Click any website to revisit it
4. Click **"Clear History"** to delete all

**KEYBOARD SHORTCUT:**
- No shortcut, but you can disable tracking in Settings

## ⚙️ SETTINGS

**HOW TO OPEN:**
Click the **⚙️ icon** at the top left

**OPTIONS AVAILABLE:**

**Dark Mode**
- ☑️ Check = Dark theme (easier on eyes)
- ☐ Uncheck = Light theme

**Track History**
- ☑️ Check = Automatically save websites you visit
- ☐ Uncheck = Don't track history

**Custom Proxy** (Advanced)
- Leave blank = Use selected proxy
- Enter custom URL if you have your own proxy

**Export Data**
- Click to save bookmarks & history as a file
- Download as JSON file

**Import Data**
- Upload previously exported JSON file
- Restore bookmarks & history

**Reset All**
- Delete everything and start fresh
- WARNING: Cannot undo!
