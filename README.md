# 🌐 ProxyHub - Advanced Web Proxy

**Live Demo:** https://6767-lgtm.github.io/proxy--aly/

A modern, feature-rich web proxy application that lets you browse websites through CORS proxy services with a beautiful UI.

## ✨ Features

✅ **Multiple Proxy Services**
- CORSProxy (default - fastest)
- AllOrigins
- Custom proxy support

✅ **Bookmarks**
- Save favorite websites
- Quick one-click access
- Custom bookmark names

✅ **History Tracking**
- Auto-tracks visited websites
- Click to revisit
- Easy management

✅ **URL Management**
- Copy URLs to clipboard
- Auto-format URLs
- Share URLs easily

✅ **Navigation**
- Back/Forward buttons
- Page refresh
- Clear interface

✅ **Customization**
- Dark mode toggle
- History tracking on/off
- Custom proxy configuration
- Settings export/import

✅ **Data Persistence**
- LocalStorage saves everything
- Works offline
- Data export/import

## 🚀 Quick Start

### Visit ProxyHub:
```
https://6767-lgtm.github.io/proxy--aly/
```

### How to Use:
1. **Enter a URL** in the input box (e.g., `google.com`)
2. **Select a proxy** from dropdown (default: CORSProxy)
3. **Click Launch** to load the website
4. **Copy URL** with the copy button
5. **Save bookmarks** for quick access

## 🔧 Technical Details

**Technology Stack:**
- HTML5
- CSS3 (Gradients, Flexbox, Animations)
- Vanilla JavaScript (No dependencies!)

**Proxy Services:**
- CORSProxy: `https://corsproxy.org/?`
- AllOrigins: `https://api.allorigins.win/get?url=`
- AllOrigins API: `https://api.allorigins.win/raw?url=`

**Browser Storage:**
- LocalStorage for bookmarks, history, preferences
- Maximum 20 history items
- Data persists across sessions

## 🌙 Dark Mode

Toggle dark mode in settings (⚙️) for a better viewing experience.

## 💾 Data Export/Import

**Export:**
- Click Settings → Export Data
- Saves as JSON file with all bookmarks and history

**Import:**
- Click Settings → Import Data
- Upload previously exported JSON file

## 🔒 Privacy & Security

⚠️ **Important:**
- Browsing goes through third-party proxy services
- Bookmarks and history stay on your computer
- Some websites may block proxy access
- Use for legitimate purposes only
- Review proxy service privacy policies

## 🐛 Troubleshooting

**Website won't load?**
- Try different proxy service from dropdown
- Check if website blocks proxies
- Verify URL is correct

**Bookmarks not saving?**
- Disable private/incognito mode
- Clear browser cache
- Check browser storage is enabled

**Copy button not working?**
- Try keyboard shortcut Ctrl+C
- Refresh page
- Try different browser
